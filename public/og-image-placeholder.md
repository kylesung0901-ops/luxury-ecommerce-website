# Open Graph 이미지 안내

이 폴더에 `og-image.png` 파일을 추가해주세요.

## 이미지 요구사항:
- 파일명: `og-image.png`
- 권장 크기: 1200 x 630 픽셀
- 형식: PNG 또는 JPG
- 용량: 1MB 이하 권장

## 이미지 생성 방법:
1. 제공하신 로고 이미지를 1200x630 크기로 리사이즈
2. 배경과 함께 브랜드 이미지로 제작
3. `public/og-image.png`로 저장

## 참고:
- 배포 후 실제 도메인 URL로 og:image URL을 업데이트해야 합니다
- Vercel 배포 후 제공되는 URL을 사용하세요
